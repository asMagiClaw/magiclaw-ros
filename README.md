# MagiClaw ROS Interface

[![ROS1](https://img.shields.io/badge/ROS1-compatible-green.svg?logo=ros&logoColor=white)](http://www.ros.org/)
[![ROS2](https://img.shields.io/badge/ROS2-compatible-green.svg?logo=ros&logoColor=white)](http://www.ros.org/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg?logo=open-source-initiative&logoColor=white)](LICENSE)

This repository provides ROS1 and ROS2 interfaces for [MagiClaw](https://github.com/asMagiClaw/magiclaw.git), allowing users to control it through standard ROS messaging protocols.

## Installation

Clone the lastest repository:

```bash
git clone https://github.com/asMagiclaw/magiclaw-ros.git
```

For ROS1, follow the [ROS1 Installation Guide](ros1/README.md), and for ROS2, follow the [ROS2 Installation Guide](ros2/README.md).
